# Smart Customer Support Portal

## Overview
Smart Customer Support Portal is a web-based platform developed by **VelsyMedia** to help businesses manage customer support efficiently. It features ticket management, knowledge base integration, and real-time communication tools for seamless customer interaction.

## Project Purpose
The portal aims to:
- Streamline customer support workflows
- Improve response times
- Enhance customer satisfaction
- Provide scalable and user-friendly support management

## Installation
Follow these steps to set up the project:

1. **Clone the repository:**
	```bash
	git clone https://github.com/VelsyMedia/smart-customer-support-portal.git
	```
2. **Navigate to the project directory:**
	```bash
	cd smart-customer-support-portal
	```
3. **Set up a Python virtual environment:**
	```bash
	python -m venv venv
	```
4. **Activate the virtual environment:**
	- On Windows:
	  ```bash
	  venv\Scripts\activate
	  ```
	- On macOS/Linux:
	  ```bash
	  source venv/bin/activate
	  ```
5. **Install dependencies:**
	```bash
	pip install -r requirements.txt
	```
6. **Run the backend server:**
	```bash
	python app.py
	```
7. **Start the frontend server (if applicable):**
	Follow frontend setup instructions in the frontend directory.

## Usage
Once both servers are running, access the portal via your web browser at `http://localhost:8000` (or the configured port).
